Create a Virtual Contest

It enable the users to run the past contests in a special mode that would imitate a real competition, currently supporting Cookoffs and LunchTime.

It uses Codechef APIs to get details about past contest and conduct it in virtual environment, giving feel as if it was hosting at real time. Try it now!

  


When you take part in virtual contests please follow those simple rules:

  • Don't participate if you've already seen the problems
  • Don't participate if you only want to commit one problem or if you don't want/can't participate in the whole contest.
  • During the contest don't read editorials, don't use somebody else's code, don't talk to other people
  • Try to make it an efficient and enjoyable training!
